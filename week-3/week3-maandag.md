## 1) Wat heb ik vandaag gedaan voor mijn project?

- we hebben deze ochtend een scrum meeting gehad en onze rollen verdeelt wie wat doet in onze groep.
- Ik heb de functionaliteiten gemaakt door scripten toe te voegen aan de html bestanden van de webshop paginas, ik heb ook wat aanpassingen gedaan aan de css zoals kleine details van de button en privacy policy.
- de favorieten pagina heb ik functioneel ook gemaakt, wanneer je op het hartje drukt dan voegt de item zichzelf toe aan de favorieten pagina, wanneer je terug op het hartje druk, verwijderd het. 
- winkelmandje is ook functioneel, als je naast de item op het winkelmandje logo drukt voegt het zich ook toe aan het winkelmandje toe. De button onderaan brengt je naar de levering pagina.
- ik heb ook de zoekbalk functioneel gemaakt, ik heb een js file gemaakt voor are producten die we hebben in onze webshop. Je kunt dus nu het product opzoeken en filteren.
- Het filteren werkt ook, ik heb het ook functioneel gemaakt, je kunt de producten filteren door prijs, beoordeling en welke de meeste of minst populair is. Deze data heb ik ook in een js file gezet zodat het kon filteren per product.
- Ik heb dan ook het winkelmandje functioneel gemaakt als u op de knop drukt om te betaling uit te voeren komt u op de levering pagina.
- Op de checkout pagina, hier moet je opties klikken, als je je leveringsmethode niet drukt kan je niet verder gaan, je bevestigt het en dan kom je naar de levering pagina.
- Op de levering pagina kan je dus je info zoals adress, e-mail,etc opschrijven en op bevestigen drukken, dan kom je op de betalings gegevens pagina.
- Wat ik gedaan heb op de betalings gegevens pagina (omdat ze fake moeten betalen), heb ik een randomised kaartnummer, naam, datum en cvc onderaan de pagina gezet die de senioren kunnen gebruiken telkens als ze een betaling willen maken. 
- Op de geslaagde betalings pagina heb ik confetti gezet en er staan 2 opties, terug te gaan naar de webshop of de bestelling te bekijken. De bestelling is dus functioneel en kun je nu terugvinden in de account pagina => onder je profiel zie je de bestelling die je hebt gedaan.
